# Welcome to Astro Modern Template

This is a showcase of the latest technologies:

- **Astro 6** with Content Layer API
- **Tailwind CSS v4** with CSS-first config
- **Nanostores** for state management
- **OXC** for lightning-fast linting

## Features

This template includes:

1. **View Transitions** for smooth navigation
2. **Dark Mode** with system preference support
3. **Type-safe** content collections
4. **Cloudflare** optimized deployment

Try toggling the theme in the header!
